AjaxSearchModalDialog

for Bootstrap

This is an improvement of Bootstrap Modal Dialog for searching things using Ajax via jQuery.

Run now!

Motivation

I have created this example to show you how I improve the modal dialog of Bootstrap for an application that I´m developing just now.

This example shows user a modal dialog to search into a JSON file a plates list that can be shorted using a custom filter dropdown list and writing the name of a plate

Github

You can download the code from my own repository in Github to be used in your project, and modify it for your porpouses.
